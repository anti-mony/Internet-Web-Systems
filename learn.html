# Property Description
1 Margin Free Area outside the content box
2 Padding Free Area inside the box from margin to content
3 Font-Weight Font Thickness
4 Color Color of the content ahead of the tag
5 Background Color Background Color of the element
6 Display Type of box/conatiner Ex: inline,block,flex
7 Float Specifies that an element should be placed along the left or right side of its container
8 Border Setting all individual border property values at once: border-width, border-style, and border-color
9 Text-Align Sescribes how inline content like text is aligned in its parent.
10 Z-Index When elements overlap, z-order determines which one covers the other. 3D for page
11 Overflow Should content clip scroll or just display in case of content getting out of parent
12 Border-Radius To add rounded corners to objects
13 Font-Family Used to definet eh font of text content
14 Width Used to specify width of element in px, % etc.
15 Height Used to specify height of element in px, % etc.
